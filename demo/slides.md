# 项目背景

::: incremental
- 格点 QCD 数值模拟 → 抽取离散有限体积能级
- 有效哈密顿量方法 → 从能级拟合提取散射相移 / 共振参数
- **NPHFforFVE.jl**：自动化有限体积哈密顿量构造 + 对角化管线
:::

# 核心能力矩阵

| 能力 | 支持范围 |
|------|----------|
| 粒子体系 | N = 2 散射态（可扩展至 N ≥ 3） |
| 粒子类型 | 可区分、全同玻色子、全同费米子 |
| 动能色散 | 相对论 / 非相对论 |
| 对称性 | O_h / C4v / C2v / C3v + 双覆盖群 |
| 同位旋 | 多物种 S_N 置换群分解, CG 系数 |
| 自旋 | 螺旋度基, Wigner D 矩阵, V_hel 构造 |
| 运动系 | Lorentz boost, kinematic factor |

# 用户工作流

```
Fock 道定义 ──→ 相互作用势 V ──→ Project 配置 ──→ compute! ──→ 能谱
```

::: incremental
1. **FockChannel** — 指定质量、自旋、同位旋、粒子统计
2. **my_V** — 用户自定义矩阵元 (CM 框架), Wigner-Eckart 约化
3. **Project / add_config!** — 设定 (L, a, Γ, n_levels)
4. **compute!** — 自动投影 + 对角化 → 能级列表
5. **输出** — 可直接与格点 QCD 能级对比拟合
:::

**势能 boost**: $V'_{fi} = \text{fac}_f \cdot V_{fi}(p^*, k^*) \cdot \text{fac}_i$

# NN 系统 (全同费米子)

**Pauli 原理**: 总波函数反对称 → 同位旋 ⊗ 置换 ⊗ 自旋

| I | κ | S | V 中交换符号 |
|---|----|---|-------------|
| 1 | [2] (对称) | 0 (反对称) | − |
| 0 | [1,1] (反对称) | 1 (对称) | + |

相互作用: 总自旋投影 $V = C \cdot (\delta_{\sigma'_1,\sigma_1}\delta_{\sigma'_2,\sigma_2} \pm \delta_{\sigma'_1,\sigma_2}\delta_{\sigma'_2,\sigma_1})$

---

# NN 运动系：结果汇总

| d | 小群 | κ=[2] (I=1,S=0) | κ=[1,1] (I=0,S=1) |
|---|------|-----------------|---------------------|
| D001 | C4v2 | A1,A2,B1,B2,E ✓ | A1,A2,B1,B2,E ✓ |
| D011 | C2v2 | A1,A2,B1,B2 ✓ | A1,A2,B1,B2 ✓ |
| D111 | C3v2 | A1,A2,E ✓ | A1,A2,E ✓ |

- NN 总自旋为整数 → 仅玻色子不可约表示非空
- d=D011 中 κ=[2]↔[1,1] 各不可约表示维数互换 (置换对称性体现)
